# Project Images Setup

## 📸 Replace Placeholder Images

Your portfolio currently uses generic Unsplash images. Replace them with actual project screenshots for better impact!

### What to Add:

1. **Take Screenshots** of your actual projects
    - Full page screenshots
    - Mobile responsive views
    - Key features in action

2. **Create GIFs** (optional but impressive)
    - Use tools like ScreenToGif or Gifox
    - Show interactive features
    - Keep file size under 5MB

3. **Save Images Here**
    - Create folder: `/public/projects/`
    - Name format: `project-name-1.png`, `project-name-2.png`
    - Use WebP format for better performance

4. **Update Projects.tsx**
    - Replace Unsplash URLs with: `/projects/your-image.png`
    - Example: `image: '/projects/restaurant-system.png'`

### Recommended Tools:

- **Screenshots**: Windows Snipping Tool, ShareX
- **GIF Recording**: ScreenToGif (Windows), Gifox (Mac)
- **Image Optimization**: TinyPNG, Squoosh
- **Mockups**: Mockuphone.com (for device frames)

### Tip:

If you don't have live projects yet, create mockups or screenshots from localhost!
